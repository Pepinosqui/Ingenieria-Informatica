# Actividad 1: Comprensión Conceptual

## 1.- ¿Cuál es la diferencia entre entrada y salida en un programa?
Las entradas son todos los datos que el programa recibe, ya sea por medio de otros programas por medio de comunicación por ejemplo, o por entradas hechas por el usuario, ya sea por medio del teclado o del mouse. Las salidas son todo lo que el programa produce como resultado y manda a otro programa o periférico, como sonido en una bocina o texto en la consola de la pantalla.

## 2.- ¿Por qué es importante declarar correctamente el tipo de una variable?
Porque no todas las variables tienen las mismas propiedades, rango de magnitud o signos. Si no se usa una variable adecuada para el uso que requiere, puede que la variable no sea lo suficientemente grande para guardar el valor deseado, deje espacio sin aprovechar, no soporte ciertas operaciones, no soporte punto decimal, no pueda tener valores negativos, entre otros.

## 3.- Explica con tus palabras qué es una clase y qué es un método.
Una clase es la forma base o plantilla para poder crear objetos, donde se definen sus miembros, variables, valores o funciones que puede realizar. Las funciones también son llamadas métodos, y son bloques de código que pueden ser ejecutados por los objetos de una clase que contenga dicho método.

## 4.- ¿Qué problema resuelve el uso de packages?
Dar un fácil acceso a clases ya creadas, ya sea que formen parte del mismo Java o sean creadas por el usuario, dando acceso a clases y métodos como si se tratase de una librería.

## 5.- Menciona dos reglas de estilo que mejoren la legibilidad del código.
Tener consistencia en como se definen las variables, dándoles nombres significativos y fáciles de entender. También tener consistencia en como se redacta el código, entrando aspectos como el uso de tabulares, donde se colocan espacios, si las llaves van en la misma línea o en otra, entre otros.
